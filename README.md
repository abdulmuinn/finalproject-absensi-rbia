# Absensi RBIA

Aplikasi Absensi Digital untuk Pengajar berbasis mobile menggunakan Flutter.

## Deskripsi Project

Absensi RBIA adalah aplikasi mobile yang membantu pengajar dalam mencatat kehadiran siswa, mengelola kelas, mencatat materi pembelajaran, melihat riwayat absensi, dan membuat rekap kehadiran secara digital.

## Fitur Utama

- Login Pengajar
- Manajemen Kelas
- Manajemen Siswa
- Absensi Siswa
- Catatan Materi Pembelajaran
- Riwayat Absensi
- Rekap Kehadiran

## Teknologi

- Flutter
- Dart
- REST API
- MySQL
